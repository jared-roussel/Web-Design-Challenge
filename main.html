Summary: Lattitude vs. Weather

Project Purpose

The purpose of this project was to analyze how weather changes as one gets closer to the equator (LAT = 0). The aim was to analyze (4) common components of weather:

  • Maximum Temperature (Fahrenheit)
  • Cloudiness (%)
  • Humidity (%)
  • Wind Speed (mph)

Project Methodology

To accomplish the project goal, we performed the following main tasks:

  1. Pulled data from OpenWeatherMap API to assemble a dataset of over 500 cities
  2. Used Matplotlib to plot the components of weather vs. the lattitude

This site provides the source data, visualizations and analysis for weather conditions.
